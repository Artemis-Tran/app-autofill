# App Autofill Extension

A powerful and customizable browser extension that automatically fills out web forms based on a provided JSON profile. It supports Shadow DOM, SPAs, and complex forms.

## Installation & Build

1.  **Clone** the repository.
2.  **Build** the extension:
    ```bash
    npm run build
    ```
3.  **Load** in browser:
    - Go to `chrome://extensions`
    - Enable **Developer mode**
    - Click **Load unpacked**
    - Select the `dist/` directory

## Usage

1.  Click the extension icon.
2.  Save your JSON profile.
3.  Click **Fill** on any form.

> **Note:** This project is under active development and APIs may change.
